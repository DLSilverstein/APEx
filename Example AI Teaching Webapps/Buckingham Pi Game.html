Matrix Rank & The Buckingham-Pi Theorem

Mastering dimensional analysis for Fluid Mechanics.

Why are we learning this?

In fluid mechanics, the Buckingham-Pi Theorem is used to simplify complex physical problems by grouping variables into dimensionless parameters (Pi terms).

Number of Pi Terms = (Total Variables, n) - (Rank of Dimensional Matrix, r)

While r is often simply the number of primary dimensions involved (like Mass, Length, Time, so r = 3), this is not always true. To guarantee you have the correct number of Pi terms, you must mathematically prove the Rank of the dimensional matrix.

Math Refresher: The Toolkit

Matrix Rank

The maximum number of linearly independent rows or columns in a matrix. In fluids, this tells us how many truly independent physical dimensions we are working with.

Determinant

A scalar value calculated from a square matrix. For a 3x3 matrix, if the determinant is NOT zero (det ≠ 0), the vectors are independent, and the rank is 3.

Submatrix (Minor)

A smaller square matrix formed by deleting entire rows and columns from the original matrix. Used to find rank when the full matrix determinant is zero.

Non-Square (3x4)

Dimensional matrices are often 3x4 or larger. The max rank is the smaller dimension (3). Check the 3x3 submatrices; if ANY has a non-zero determinant, rank = 3.

Walkthrough: Finding the Rank

Let's find the rank of this classic 3x3 matrix.

Step 1: Check the 3x3 Determinant

det(A) = 1(5×9 - 6×8) - 2(4×9 - 6×7) + 3(4×8 - 5×7)
det(A) = 1(45 - 48) - 2(36 - 42) + 3(32 - 35)
det(A) = 1(-3) - 2(-6) + 3(-3)
det(A) = -3 + 12 - 9 = 0

Because the determinant is zero, the rows are linearly dependent. The rank is NOT 3.

Step 2: Check 2x2 Submatrices (Minors)

We step down. Is the rank 2? We check the 2x2 submatrices. Let's pick the top-left one:

det = (1 × 5) - (2 × 4) = 5 - 8 = -3

Because this 2x2 determinant is non-zero (-3), we have found a set of linearly independent vectors of size 2.

Therefore, the Rank of this matrix is 2.
